Learn Computing Activity
I chose to follow the Command Line activity, because I didn't get in as much practice in class as I would have hoped. I've worked in terminal before, but understanding how it actually works has always eluded me. The command line receives instructions via text from the user and tells the operating system to run them once the user presses enter. You can navigate files, create new directories, and explore your computer in this text-based application.

Queering the Map

This is a digital project I was introduced to by Dr. Sample in my Intro to Digital Studies class. Ever since then, it's my go-to digital project. What's so interesting about this project is the collaboration aspect as well as the contirbutors' ability to comment anonymously. You can zoom in on almost any part of the world and read short stories or anecdotes written by people from the LGBTQIA+ community. Not only do I think this allows people to be seen, but I think it lets people know they are not alone.

  • One of the best things about this project is the Davidson contribution. There is one point that is clearly in Belk dorm.
  • I also love this project because of that idea that people writing these stories might not have anyone else they feel comfortable sharing their stories with, but they trust Queering the Map.

Lili Elbe Digital Archive

This is a project I studied over this past summmer in an online class. Not only did I not know who Lili Elbe was (one of the first people to go through a sex-change surgery) before studying this project, but I'd never seen a text comparison like the one this project assembles. The Collation Studio allows users to compare different versions of Elbe's text (describing her transition), right next to one another.

  • Not only can you compare these documents, but this website makes editions of Elbe's books digitally available that might otherwise be kept hidden in the shelves of a library. This website represents what a lot of digital archivistis aim to do: make resources accessible to anyone with an internet connection and a computer

Jeopardy! Archive

The final project I want to highlight is less about digital archives or activism, but instead deals with Jeopardy! I ran trivia for Davidson last year and I used this resource all the time for trivia categories and other ideas. This website is a record of almost all of the Jepoardy games and their questions going all the way back to Season 1, which premiered in 1985.

  • You can choose the season and then a specific game to see who hosted that day, the circumstances of the game, and the scores of each contestant (plus their names) at the end of each round.
  • The site is also always being updated to include current games. The most recent game to be included is from August 13, 2021!
  • I love the fact that this site is assembled by Jeopardy! fans and you can sometimes seen the "homegrown" aspects of the site.
